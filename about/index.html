氢商店

Version 1.1.1 (Hydrogen)

氢商店 (Hydrogen Store) 是一个开源、轻量级的静态应用分发平台。


它基于 Material 3 设计语言构建，旨在提供、流畅且现代化的用户体验。本项目无需后端服务器，所有数据均通过 JSON 文件驱动。


主要特性：

  • 纯静态部署
  • 深色模式自动适配 & 自定义
  • PWA 级单页应用体验
  • 新增应用 JSON 生成器
  • 3A 网页：AI, Android, Amazing(
